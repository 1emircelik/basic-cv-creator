# CV Builder (ATS Uyumlu) - HTML/JS

Bu proje tamamen tarayıcıda çalışır. CV formunu doldur → **PDF İndir** (tarayıcının yazdır/PDF özelliği).

## Emoji / Karmaşık içerik
Şablon tek kolon ve ATS uyumlu düz metin odaklıdır. Emoji veya karmaşık semboller eklemeyin.

## Kurulum (lokal)
Node.js / npm gerektirmez.

1) Bu klasörü aç: `cv-builder/`
2) `index.html` dosyasını tarayıcıda açın.

Ayrıntı: `README_DEPLOY.md` dosyasına bak.

## PDF Alma
- Tarayıcıdaki **PDF İndir** butonuna basın.
- Sisteminizin yazdırma ekranında hedefi **PDF olarak kaydet** olarak seçin.

## Dosya yapısı
- `index.html` : Form + önizleme
- `style.css` : Stil ve print düzeni
- `app.js` : Formdan veriyi alır, önizlemeyi günceller
